Format

6.3s Seamless Loop

Frame-perfect loop, split-sentence open. Viewers rewatch 2–3x without noticing, pushing watch time over 100%.

14–16s Reel

The completion sweet spot. Hardest length to fail on completion rate: the biggest discovery signal.

30s Demo

Show-and-tell. Product on screen frame 1. Use for YT Shorts and Shop-tagged posts.

Content Path

Path A · Storytelling Pitch

Hook → agitate → reveal → proof → CTA. One verbal hook, one CTA. 15–35s.

Path B · Real-Life Use

Loopable lifestyle content, product incidental. CTA lives in the caption. Awareness and trust lane.

Path C · Show-and-Tell

Demo removes doubt. Product on screen from frame 1, benefit spoken over it.
